项目介绍
Whale-LLT App版本是一个基于 Web 技术构建的本地化 AI 互动工具。

运行方式：

Android：已打包为 APK，安装即可使用，支持系统级消息通知。

PC/Local：支持直接双击 index.html 运行，数据存储于本地浏览器数据库 (IndexedDB)。

功能特性
基于贡献者开发日志整理：

核心通讯：

消息全流程（发送/接收/撤回/编辑）。

语音消息与电话功能全流程（含 ASR/TTS）。

群聊功能及多人互动。

红包功能全流程（发红包/抢红包/记录）。

社交互动：

朋友圈（支持九宫格、评论、识图）。

论坛/微博功能（支持发帖、评论、回复）。

一起听歌功能。

个性化与资源：

表情包管理（保存/发送/导入）。

头像上传与角色构建。

气泡界面全流程（支持导入导出、分组）。

置顶功能与界面构建优化。

数据与设置：

完整的数据导出/导入流程。

API配置保存与手动输入模型。

网页版主题色修改与夜间模式。

全局记忆与识图功能接入。
